Le but de cet excercice est de réaliser une calculette

capable de :

* Effectuer un calcul simple : 9 + 2.

* Faire des calculs à la chaîne, par exemple : 9 + 2 + ... ; lorsqu'on clique à nouveau sur un opérateur, il faut afficher le résultat du calcul précédent.

* Gérer l'exception d'une division par 0 !

* Vérifiez que le clic sur 1 puis 2, donne 12 et non 2.

* Le texte saisi par l'utilisateur doit être converti en nombre. Par exemple pour convertir un String en un double nb=Double.ParseDouble("12" ) donne nb=12.

* Il est impératif que la virgule n’apparaît qu’une seule fois dans un nombre (Faux si 12…3)

* Donner la possibilité de tout recommencer à zéro.
